Proyek Unggulan

Daftar implementasi sistem kendali, robotika, dan perangkat pintar yang telah saya kembangkan:

ROBOTIKA & SISTEM TERTANAM

Prototype Smart Vacuum Cleaner Robot Berbasis Arduino Uno

Robot penyedot debu otomatis pintar yang bergerak secara otonom. Mengintegrasikan Unit Input (saklar daya & sensor ultrasonik), Unit Kontrol (Arduino UNO & Motor Shield), serta Unit Output (motor DC vakum, gear motor penggerak roda, dan motor servo pengarah sensor). Setelah saklar aktif, robot secara otomatis menyedot debu lantai sambil memetakan rintangan di sekitarnya guna menentukan arah gerak hindaran terbaik secara real-time.

Arduino UNO Sensor Ultrasonik Motor Shield Servo MG995
OTOMASI & ANTARMUKA (GUI)

Sistem Pakan Otomatis Berbasis MATLAB GUI dan Arduino Uno

Rancangan sistem pemberian pakan otomatis terjadwal untuk meningkatkan efisiensi peternakan. Menggabungkan sensor LDR untuk mendeteksi intensitas cahaya dan motor servo MG995 sebagai penggerak mekanis katup wadah pakan. Dilengkapi fitur Mode Otomatis berbasis jadwal/cahaya serta Mode Manual yang dikendalikan langsung secara real-time melalui visual antarmuka interaktif MATLAB GUI.

MATLAB GUI Arduino Uno Sensor LDR Servo MG995
SISTEM KENDALI INDUSTRI

Robot Power Control & Heater Monitoring System

Proyek optimalisasi kelistrikan mesin industri yang dikembangkan selama masa magang. Berfokus pada perancangan kendali distribusi daya untuk lengan robot mekanis, serta membangun sistem pemantauan beban pemanas (heater load monitoring) guna mendeteksi anomali arus elektrik secara dini untuk mencegah kegagalan sistem.
